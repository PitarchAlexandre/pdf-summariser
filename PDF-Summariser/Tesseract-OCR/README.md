# Installing Tesseract OCR

## English Version

1. Download Tesseract OCR from the following link:  
   https://digi.bib.uni-mannheim.de/tesseract/

2. Install it on your machine.

3. Once the installation is complete, go to the following folder:

`C:\Users\<YourUsername>\AppData\Local\Tesseract-OCR`

4. Copy or move all the files and folders from this repository and paste them into the `Tesseract-OCR` directory.
