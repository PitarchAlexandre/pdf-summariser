# Installation de Tesseract OCR

## Version française

[README in english here](README.md)

1. Téléchargez Tesseract OCR depuis ce lien :
   https://digi.bib.uni-mannheim.de/tesseract/

2. Installez-le sur votre machine.

3. Une fois l’installation terminée, rendez-vous dans le dossier suivant :

`C:\Users\<VotreNomUtilisateur>\AppData\Local\Tesseract-OCR`

4. Copiez ou déplacez tous les fichiers et dossiers présents dans ce dépôt, puis collez-les dans ce dossier `Tesseract-OCR`.
