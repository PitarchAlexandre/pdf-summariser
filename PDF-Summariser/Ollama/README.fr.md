# Installation d’Ollama

## Version française

[README in english here](README.md)

1. Téléchargez Ollama depuis le site officiel :
   https://ollama.com/

2. Installez-le sur votre machine.

3. Une fois l’installation terminée, rendez-vous dans le dossier suivant :

`C:\Users\<VotreNomUtilisateur>\AppData\Local\Programs\Ollama`

4. Copiez ou déplacez tous les fichiers et dossiers présents dans ce dépôt, puis collez-les dans ce dossier Ollama.
