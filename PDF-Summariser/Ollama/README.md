# Installing Ollama

## English version

[README en francais ici](README.fr.md)

1. Download Ollama from the official website:
   https://ollama.com/

2. Install it on your computer.

3. Once the installation is complete, navigate to the following folder:

   `C:\Users\<YourUsername>\AppData\Local\Programs\Ollama`

4. Copy or move all files and folders from this repository and paste them into the Ollama directory.
